# Responsive E-Ticaret Arayüzü 
Bu proje, HTML ve CSS kullanılarak geliştirilmiş tek sayfalık bir e-ticaret arayüzüdür. Amaç; modern CSS tekniklerini kullanarak Grid, Flexbox, Responsive Tasarım, Etkileşim Durumları ve CSS Değişkenleri konularını uygulamalı olarak göstermektir.

## Öğrendiklerim / Zorlandıklarım

-Grid ve Flexbox birlikte kullanım stratejileri
-Responsive tasarımda breakpoint planlaması
-Hover ve focus durumlarının UX üzerindeki etkisi
-CSS değişkenleri ile daha sürdürülebilir stil yönetimi
-Kart layout’larında dikey hizalamanın önemi
